Enabling iFAX Support Access

Should you require assistance configuring your HylaFAX Enterprise Appliance, we will need to start a remote connection to your server in order to help. Before initiating the connection, please contact us at support@ifax.com for a “Support Tunnel ID.”

Once you receive the Support Tunnel ID, you must access the HylaFAX Enterprise Appliance console and select “Advanced Options” -> “Enable iFAX support access.”

Enter the ID supplied to you in the “Support Tunnel ID” field, then select the "Enable" button and press ENTER. This creates a secure connection to our servers where we will be able to access your HylaFAX Enterprise server remotely and provide assistance.
