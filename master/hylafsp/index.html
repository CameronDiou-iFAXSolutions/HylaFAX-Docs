    HylaFSP

    HylaFSP Quickstart Guide 1. Summary

    HylaFSP is an extremely lightweight client for Microsoft Windows and Apple macOS that offers a tightly integrated and completely native faxing experience. It’s perfect for non-technical users and allows you to fax from any application you can print from. It even integrates with Windows/Outlook and macOS Address Books!

    For more information, visit us online:

    https://www.ifax.com/products/hylafsp/

    1.1. Microsoft Windows System requirements • Windows 7, 8.1, or 10 (requires Workstation License) • Windows Server 2008, 2012, 2016 (requires Server License)

    1.2. Apple macOS System requirements • macOS 10.7 - 10.13 (requires Workstation License)

    1. Installation

    2.1. Windows Install

    Here are the steps for installing HylaFSP on the Windows platform:

    1. Download HylaFSP

    https://www.ifax.com/software-downloads/

    1. Double click on the HylaFSP setup.exe file and follow the on screen instructions

    2. Open HylaFSP from the Windows Control Panel and do the following:

      a. Click on the "Server" tab and enter your fax server’s IP address b. Click on the "User Defaults" tab and enter your HylaFAX username, password, and your email address in the respective fields c. Click OK

    3. Restart MS Fax Services (or reboot the PC)

    4. To print to fax, open a Notepad or Word document and select Print from the File menu

    5. Select the "Fax" printer and click OK

    6. When the Windows Fax & Scan wizard appears, select "Connect to a fax modem" and then choose all of the recommended options.

    7. In the next window that appears, enter the destination fax number and click Send

    ***Info:

    HylaFSP is a virtual modem for Windows Fax Services. Therefore, during the initial configuration of Windows Fax & Scan, you’ll want to select "Connect to a fax modem" instead of "Connect to fax server".
